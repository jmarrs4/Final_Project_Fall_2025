    Overall Findings

    After combining all of the data into a single data set there is a negative correlation between crime and temperature as well as UV Index. However, because of mild weather cities like LA an entire data set can be shifted becasue of sample size. In conclusion, we believe that there is still a positive correlation between crime and the weather, but you would need more data with a larger sample size of different weather conditions.
